# DocuMind-AI-RAG-Based-Document-Intelligence-Assistant
DocuMind AI — An intelligent RAG-powered assistant that transforms PDFs into dynamic, queryable knowledge bases with semantic search and context-aware answers.

---

##  Features
- **PDF Ingestion** — Downloads and processes PDFs automatically.
- **Text Chunking** — Splits documents into manageable chunks.
- **Semantic Search** — Embeddings with `sentence-transformers/all-MiniLM-L6-v2`.
- **Vector Store** — Stores embeddings in **ChromaDB**.
- **Context-Aware Responses** — Powered by **LangChain** + **Groq LLM**.
- **Interactive UI** — Gradio-based chatbot for querying documents.

---

## Tech Stack
- **Python**
- **LangChain** (core orchestration)
- **LangChain-Unstructured** (document loader)
- **LangChain-Chroma** (vector store)
- **LangChain-Groq** (LLM integration)
- **HuggingFace Embeddings**
- **Gradio** (UI)
